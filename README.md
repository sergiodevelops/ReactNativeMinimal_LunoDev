## Install dependencies and run 

```bash
# with YARN
yarn
```
```bash
# or with NPM
npm install
```

## Run for development 

### Run for Android or iOS platform

```bash
# with YARN
yarn start
```
```bash
# or with NPM
npm run start
```

### Run for Web server platform

```bash
# with YARN
yarn web
```
```bash
# or with NPM
npm run web
```
